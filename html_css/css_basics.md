# Selecionando itens 

## ID (devem ser únicos)

```css
#nomeid {
    color: blue;
}
```

## Classe (devem ser usados para quando vários elementos tem os mesmos atributos)
```css
.nome_classe {
    color: blue;
}
```

# Atributos interessantes 
```css
div {
    text-transform: uppercase;
    border: 3px solid black;
}
```
