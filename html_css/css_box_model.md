# Como fazer 

## Gerais 
```css
h1{
    background: red;
    margin: 0; /* Pode ser negativa pra uma coisa entrar na outra */
    padding: 20px 10px 40px 20px;
    border-style: solid;
    border-width: 50px;
    border-color: green;
    border: 5px solid black;
    border-bottom-style: dashed;
    border-radius: 5px;

}
```

```css

    display: inline;
```
