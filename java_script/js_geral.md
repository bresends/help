
# Print com variáveis (Template String)
```javascript
window.alert('Holá ' + nome + '!')
window.alert(`Holá ${nome}!")
```

# Conversão de tipos 
```javascript
// Forçando Tipo
Number.parseInt(n)
Number.parseFloat(n)

// Deixando Escolher
Number(n)
String(n)
n.toString()
```

# Métodos de Strings

```javascript
str.LowerCase()
```

# Usando comparadores 

Não use '==' nem '!=' em js
